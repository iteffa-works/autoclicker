# Автоклікер і макроси від Flowaxy Digital Studio

Desktop-застосунок для Windows (Python 3.12+; розробка під 3.14): автоклікер (простий миша, послідовність клавіш/миші, повтор однієї клавіші), запис/відтворення макросів, глобальні хоткеї (Win32 RegisterHotKey або pynput), тест клавіатури/миші, трей, оверлей статусу, темна/світла тема.

- **Один екземпляр:** повторний запуск звертає вікно вже запущеного процесу.
- **Закриття (×):** за замовчуванням вихід з програми. Опція «Закривати в трей замість виходу» змінює це.
- **Згортання:** мінімізація в трей (якщо увімкнено «Мінімізація в трей»). На Windows 11 іконка може потрапити в «приховані» іконки — відкрийте список і закріпіть потрібну програму.
- **Глобальні клавіші:** режим «Авто» спочатку реєструє **Win32** (`RegisterHotKey`), інакше — **pynput**. У повноекранних іграх з ексклюзивним вводом або античитом хоткеї ОС можуть не спрацьовувати — спробуйте режим вікна без рамки (borderless) або інші бинди. Бинди: перемикач автоклікера, пауза, перемикач відтворення макросу, перемикач запису макросу, перемикач трею (лише з «Мінімізація в трей»), аварійна зупинка.

## Залежності

```text
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

Для збірки додатково: `pip install pyinstaller` або `pip install -e ".[dev]"` з `pyproject.toml`.

## Запуск з коду

```text
python main.py
```

## Збірка одного портативного `.exe` (onefile)

```text
python -m PyInstaller --clean --noconfirm autoclicker.spec
```

Якщо збірка падає з **`PermissionError: ... dist\autoclicker.exe`** — процес **autoclicker.exe** ще запущений (або файл тримає Провідник/антивірус). Закрийте програму з трею та повторіть, або з кореня проєкту: `powershell -ExecutionPolicy Bypass -File .\scripts\build_exe.ps1` (скрипт спробує завершити процес перед збіркою).

Іконка на файлі `.exe` з’являється лише після **повної** перезбірки. Якщо в Провіднику все ще стара картинка — перезапустіть Провідник або ПК (кеш іконок Windows). За потреби перегенеруйте `assets/icons/favicon.ico` з PNG: `python scripts/rebuild_favicon_ico.py` (потрібен Pillow).

Готовий файл: `dist\autoclicker.exe`. Копіюйте на інший ПК без встановлення Python. Налаштування та макроси зберігаються поруч із `.exe` у папках `config\` та `data\`.

## Структура даних

| Шлях | Призначення |
|------|-------------|
| `assets/icons/favicon.png`, `assets/icons/favicon.ico` | Іконка вікна, трею та збірки `.exe` |
| `assets/images/character.png` | Декоративне зображення в лівій панелі головного вікна |
| `config/settings.json` | Налаштування (створюється автоматично) |
| `config/plugin.json` | Метадані студії та версія (дублює дані в коді) |
| `data/macros/*.json` | Збережені макроси |

Приклади: `config/settings.example.json`, `data/macros/example_macro.json`.

## Права та безпека

Використовуйте лише на власному ПК у законних сценаріях. Застосунок не використовує драйвери ядра та не призначений для обходу захисту ігор чи інших систем.
